Proceso

Cómo funciona

Cuatro pasos entre tu fuente de datos y un proyecto Django listo para ejecutar.

01
Conecta tu fuente
Pega una URL de API pública o arrastra un archivo JSON, CSV o XML. WebBuilder lo descarga y parsea automáticamente.
02
La IA analiza el schema
El LLM seleccionado detecta los campos, infiere tipos y decide cuáles mostrar. Tú ajustas campo a campo antes de continuar.
03
Genera el proyecto
WebBuilder genera el proyecto Django completo: modelos, vistas, templates y CSS adaptados al tipo de datos detectado.
04
Despliega con n8n
El flujo automatizado de n8n empaqueta el proyecto, construye la imagen Docker y lo despliega en un puerto disponible.
